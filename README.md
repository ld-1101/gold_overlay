# 🏅 积存金 - 桌面悬浮窗

实时显示国内金价（沪金主力）的 Windows 桌面悬浮窗，半透明玻璃效果，可拖动，自动刷新。

## 功能

- **国内金价** — 沪金主力合约，单位 元/克
- **透明悬浮** — 半透明暗色背景，不遮挡桌面
- **可拖动** — 拖拽标题栏任意移动位置
- **自动刷新** — 每 10 秒更新，数据源新浪财经
- **极简设计** — 只有一个关闭按钮

## 启动方式

- 双击 `积存金.exe` 直接运行
- 或双击 `启动悬浮窗.bat`

## 自定义

右键窗口 → 刷新 / 置顶 / 退出。如需修改自选列表，编辑源码 `gold_overlay.py`。

## 技术

- Python 3.12 + tkinter
- PyInstaller 打包为单文件 EXE
- 新浪财经 `nf_AU0` 接口
